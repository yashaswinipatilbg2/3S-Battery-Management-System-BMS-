3S Battery Management System (BMS)

* Project Overview

  * Design of a 3-cell (3S) Lithium-ion Battery Management System
  * Uses BQ77915 protection IC
  * Ensures safe and reliable battery operation

* Key Features

  * Overcharge protection
  * Over-discharge protection
  * Short-circuit protection
  * Cell voltage monitoring
  * Reliable battery safety design

* Project Files

  * 3S_BMS_Schematic_BQ77915.pdf – circuit schematic
  * 3S_BMS_Gerber_Files.zip – PCB manufacturing files
  * PCB_3D_View.png – PCB 3D view

* Tools and Technologies

  * EasyEDA for schematic and PCB design
  * BQ77915 battery protection IC
  * Embedded electronics design concepts

* Applications

  * Battery packs
  * Portable electronics
  * Small EV systems
